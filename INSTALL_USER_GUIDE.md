# 📦 카카오 RSU 자산 위젯 설치 및 사용 가이드

이 문서는 개발자가 아닌 일반 사용자도 쉽게 따라할 수 있도록, 카카오 RSU 자산 위젯 크롬 확장 프로그램의 설치와 사용법을 친절하게 안내합니다.

---

## 1. 설치 방법

### 1-1. 크롬 웹스토어에서 설치 (권장)

1. 크롬 브라우저를 엽니다.
2. [Chrome 웹스토어](https://chrome.google.com/webstore/category/extensions)에서 "카카오 RSU Value Tracker"를 검색합니다.
3. 확장 프로그램 페이지에서 **[Chrome에 추가]** 버튼을 클릭합니다.
4. 설치가 완료되면, 크롬 우측 상단에 노란색 카카오 아이콘이 나타납니다.

> **참고:** 아직 웹스토어에 등록되지 않았다면, 아래의 수동 설치 방법을 따라주세요.

### 1-2. 수동 설치 (압축파일로 설치)

1. 제공받은 압축파일(.zip)을 내 컴퓨터에 저장합니다.
2. 압축을 풀어 폴더로 만듭니다.
3. 크롬 브라우저에서 주소창에 `chrome://extensions` 를 입력해 이동합니다.
4. 우측 상단의 **[개발자 모드]** 스위치를 켭니다.
5. **[압축해제된 확장 프로그램을 로드합니다]** 버튼을 클릭합니다.
6. 2번에서 만든 폴더를 선택합니다.
7. 설치가 완료되면, 크롬 우측 상단에 아이콘이 나타납니다.

---

## 2. 사용 방법

### 2-1. RSU 정보 입력

1. 크롬 우측 상단의 카카오 RSU 아이콘을 클릭합니다.
2. 팝업 창이 열리면, **보유한 RSU 수량**과 **평균 매입가**를 입력합니다.
3. [저장] 버튼을 누르면 입력한 정보가 저장됩니다.

### 2-2. 위젯 사용

- 모든 웹사이트 우측 하단에 자산 위젯이 자동으로 표시됩니다.
- 위젯에는 현재 카카오 주가, 내 RSU 자산, 수익률, 목표 달성률 등이 표시됩니다.
- 위젯 헤더를 마우스로 드래그하여 원하는 위치로 옮길 수 있습니다.
- [−] 버튼으로 위젯을 접을 수 있고, [+] 버튼으로 다시 펼칠 수 있습니다.
- [×] 버튼으로 위젯을 숨길 수 있습니다. (다시 보려면 팝업에서 위젯 표시를 켜세요)
- 위젯의 위치와 접힘/펼침 상태는 자동으로 저장되어, 새로고침해도 그대로 복원됩니다.

### 2-3. 목표 달성 알림

- 내 자산이 목표(예: 테슬라 Model 3, 유럽 왕복 항공권 등)를 100% 달성하면, 크롬 알림으로 축하 메시지가 자동으로 표시됩니다.
- 알림을 받고 싶지 않다면, 팝업의 [알림 활성화] 체크를 해제하세요.

## 3. 자주 묻는 질문 (FAQ)

**Q. 위젯이 안 보여요!**
- 팝업(아이콘 클릭)에서 [위젯 표시]가 켜져 있는지 확인하세요.
- 페이지를 새로고침하거나, 다른 웹사이트에서도 확인해보세요.

**Q. 주가가 갱신되지 않아요!**
- 인터넷 연결 상태를 확인하세요.
- 팝업을 열면 자동으로 최신 주가가 갱신됩니다.
- 시장이 닫힌 시간(평일 09:00~15:30 외)에는 가격이 변동되지 않을 수 있습니다.

**Q. 내 정보가 외부로 전송되나요?**
- 아니요! 모든 정보는 내 브라우저(Chrome)에만 저장되며, 외부 서버로 전송되지 않습니다.

**Q. 위젯 위치/상태가 초기화돼요!**
- 크롬의 저장소 권한이 제한된 경우(시크릿 모드 등)에는 저장이 안 될 수 있습니다.
- 일반 모드에서 사용해 주세요.
