# 📦 영차영차 설치 가이드

## 🚀 개발자 모드로 설치하기

### 1단계: 파일 준비

현재 폴더에 다음 파일들이 있는지 확인하세요:

```
├── manifest.json
├── popup.html
├── popup.css
├── popup.js
├── background.js
├── content.js
└── icons/
    ├── icon16.png
    ├── icon32.png
    ├── icon64.png
    └── icon128.png
```

### 2단계: Chrome 브라우저에서 익스텐션 로드

1. **Chrome 브라우저 열기**

   - Google Chrome을 실행합니다

2. **확장 프로그램 페이지 접속**

   - 주소창에 `chrome://extensions/` 입력하여 접속
   - 또는 메뉴 → 도구 더보기 → 확장 프로그램

3. **개발자 모드 활성화**

   - 우측 상단의 "개발자 모드" 토글 버튼 클릭
   - 버튼이 파란색으로 변하면 활성화됨

4. **확장 프로그램 로드**

   - "압축해제된 확장 프로그램을 로드합니다" 버튼 클릭
   - 이 프로젝트 폴더 선택 (manifest.json이 있는 폴더)

5. **설치 완료 확인**
   - 영차영차가 확장 프로그램 목록에 나타남
   - 브라우저 툴바에 📈 아이콘이 표시됨

### 3단계: 초기 설정

1. **RSU 정보 입력**

   - 툴바의 📈 아이콘 클릭
   - RSU 보유 수량 입력 (예: 125)
   - 평균 매입가 입력 (선택사항, 예: 85000)
   - "저장" 버튼 클릭

2. **설정 확인**

   - 위젯 표시 체크박스 확인 (기본: 활성화)
   - 알림 활성화 체크박스 확인 (기본: 활성화)
   - 목표 주가 설정 (선택사항)

3. **테스트**
   - 임의의 웹페이지 방문
   - 우측 하단에 RSU 위젯이 표시되는지 확인
   - 위젯이 현재 주가와 자산 정보를 보여주는지 확인

## 🔧 트러블슈팅

### 아이콘이 보이지 않는 경우

1. `chrome://extensions/`에서 영차영차가 활성화되어 있는지 확인
2. 브라우저를 완전히 종료 후 재시작
3. 확장 프로그램을 제거 후 다시 설치

### 위젯이 표시되지 않는 경우

1. 팝업에서 "위젯 표시" 설정이 체크되어 있는지 확인
2. 페이지를 새로고침
3. 다른 웹사이트에서 테스트

### 주가 정보가 로딩되지 않는 경우

1. 인터넷 연결 상태 확인
2. 개발자 도구(F12) → Console 탭에서 오류 메시지 확인
3. 팝업에서 "새로고침" 버튼 클릭

### 설정이 저장되지 않는 경우

1. Chrome의 저장소 권한 확인
2. 시크릿 모드에서는 일부 기능이 제한될 수 있음
3. 다른 Chrome 프로필에서 테스트

## 🎯 주요 기능 테스트

### 1. 주가 조회 테스트

- 팝업 열기 → "새로고침" 버튼 클릭
- 현재가가 업데이트되는지 확인

### 2. 자산 계산 테스트

- RSU 수량: 100주 입력
- 현재가가 90,000원일 때
- 총 자산이 9,000,000원으로 표시되는지 확인

### 3. 위젯 기능 테스트

- 위젯 드래그하여 위치 이동
- 최소화 버튼(-) 클릭
- 닫기 버튼(×) 클릭

### 4. 알림 테스트

- 목표 주가를 현재가보다 낮게 설정
- 알림이 발생하는지 확인

## 📱 권한 설명

확장 프로그램 설치 시 요청되는 권한들:

| 권한             | 용도         | 필요한 이유                          |
| ---------------- | ------------ | ------------------------------------ |
| storage          | 설정 저장    | RSU 수량, 매입가 등 사용자 설정 저장 |
| notifications    | 알림 표시    | 목표가 도달 시 알림                  |
| alarms           | 주기적 실행  | 10분마다 주가 자동 업데이트          |
| activeTab        | 현재 탭 접근 | 위젯 표시를 위한 페이지 접근         |
| host_permissions | 외부 API     | Yahoo Finance API 호출               |

## 🔒 개인정보 보호

- **로컬 저장**: 모든 데이터는 Chrome 로컬 저장소에만 저장
- **외부 전송 없음**: RSU 수량, 매입가 등 개인정보는 외부로 전송되지 않음
- **API 사용**: 주가 정보만 Yahoo Finance API에서 조회
- **암호화**: Chrome의 기본 저장소 암호화 사용
